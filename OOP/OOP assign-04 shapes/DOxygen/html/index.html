Program Introduction

My World Famous Paintings

What is this program can do?

This project is a simple shape calculator. It has a parent class called Shape, which is then inherited by two child classes called Circle and Square. Users can customize the color of the shape. Besides that, User will be askd to enter ridus and side-length then the program can calculate the circumference and area, shows it to users.

What are skill features used in this program?

  1. Practice of inheritance
  2. Pure virtual functions, override
  3. Dynamic memory (new,delete)
